# 关于导数几何意义的数学可视化

## 简介
本课件使用Geogebra与Manim对导数的几何意义进行了数学可视化，并在Geogebra中提供了可交互模块。

## 展示的知识点

* 导数的几何意义

## GeoGebra 交互功能简介

![bb6d7f1e-1bff-495e-b561-b67b4bf45467.png](https://img.remit.ee/api/file/BQACAgUAAyEGAASHRsPbAAEMr3VpQAJ5GKL4Y1HriW2MbRNH5nWcvgAC5iUAAismAAFW_BUTpK6gubM2BA.png)

如图所示，我们在GeoGebra中提供了多个可交互模块：

* ε、δ、h、dx滑动条

自由控制各变量数值以观察它们对导数的影响。


* “演示极限过程”按钮

按下后使h从1逐渐变化至0，从而演示极限逼近的过程。

* “重置”按钮

按下后将h重置为1。

* “函数f(x)=”输入框

在这里输入不同函数可以改变演示的函数。

## 开发工具

* Python 3.12.12
* Manim v0.19.1
* GeoGebra Calculator 6.0.9051

## 选择这些工具的理由
Manim与GeoGebra都是数学可视化中常用的工具，操作简单，易于上手。考虑到仅使用GeoGebra制作可视化效果不佳，我们额外使用Manim制作了一个演示动画。
